# AI Chatbot (Next.js)

A modern AI chatbot interface built using **Next.js, React, and Tailwind CSS**.
This project demonstrates how to build a scalable web-based chat interface that can integrate with large language models such as **Ollama** or other AI APIs.

## Features

* Modern chat interface
* Built with Next.js App Router
* Responsive UI using Tailwind CSS
* Ready for AI model integration
* Easily deployable on cloud platforms

## Tech Stack

* Next.js
* React
* TypeScript
* Tailwind CSS
* Node.js

## Project Structure

```
ai-chatbot
│
├── app/            # Main application pages
├── public/         # Static assets
├── package.json    # Project dependencies
├── next.config.ts  # Next.js configuration
└── tsconfig.json   # TypeScript configuration
```

## Installation

Clone the repository:

```
git clone https://github.com/yashrajagawane/AI-Chatbot.git
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```

Open in browser:

```
http://localhost:3000
```
